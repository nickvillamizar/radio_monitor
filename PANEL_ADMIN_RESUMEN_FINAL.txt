                           🔧 API ENDPOINTS (TÉCNICO)
═══════════════════════════════════════════════════════════════════════════════════

GET /api/emisoras
├─ Retorna lista de TODAS las emisoras
├─ Status: 200 OK
└─ Incluye: id, nombre, país, URL, estado, plays_24h, etc.

GET /api/emisoras/<id>  [NEW]
├─ Retorna detalles de UNA emisora
├─ Status: 200 OK o 404 NOT FOUND
└─ Incluye: todos los campos + plays_7d calculados

POST /api/emisoras      [ENHANCED]
├─ Crea nueva emisora
├─ Status: 201 CREATED
├─ Campos: nombre*, país*, url_stream*, sitio_web (opcional)
└─ Valida: URL única, formato válido

PUT /api/emisoras/<id>  [ENHANCED]
├─ Actualiza emisora existente
├─ Status: 200 OK
├─ Campos: cualquiera (todos opcionales)
└─ Valida: existencia, URL única

DELETE /api/emisoras/<id>
├─ Elimina emisora definitivamente
├─ Status: 200 OK
└─ Retorna confirmación
