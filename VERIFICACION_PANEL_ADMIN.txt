                    2️⃣  ESTADÍSTICAS EN TIEMPO REAL
═══════════════════════════════════════════════════════════════════════════════════

El grid superior debe mostrar 4 números:

[  ] "Total de Emisoras" → Muestra un número (11 o similar)
[  ] "Activas (24h)" → Muestra un número (0-11)
[  ] "Inactivas" → Muestra un número
[  ] "Total de Plays" → Muestra un número (ej: 694)

Verificación: Total + Inactivas = Activas (ej: 4 activas + 7 inactivas = 11 total)

[  ] Las sumas coinciden correctamente
